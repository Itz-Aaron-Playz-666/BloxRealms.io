# BloxRealms.io
## About
  **BloxRealms.io** Is A Game That Allows Players To Interact With A World That Is Full Of Mysteries.
## Core Logics
  **BloxRealms.io** Doesn't Use **Pre-Generated Maps**,It Generates A New Map Every Time Players Create A World.This Game Is Coded Through **C#**.
Game Engines Such As Unity Isn't Used During Game Development And Design.
